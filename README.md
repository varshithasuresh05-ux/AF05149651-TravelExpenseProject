Travel Expense Tracker (Java + MySQL)

## Description
A console-based Java application to manage travel expenses with:
- User authentication
- Trip management
- Expense tracking
- Search functionality
- CSV report export
- Analytics

## Features
- Add / View Trips
- Add / View Expenses
- Search by category
- Export report to CSV
- View total expense
- Analytics (top spending category)

## Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE

## Database Setup
1. Open MySQL Workbench
2. Run `sql/schema.sql`

## How to Run
1. Open project in Eclipse
2. Configure DBConnection.java
3. Run `Main.java`

## Sample Login
Register a user first, then login.

## Output
- Console-based interaction
- CSV report generated
